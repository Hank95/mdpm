# Project Instructions

## Project Tracking
This project uses a markdown-based task tracking system. See `.claude/rules/project-tracking.md` for full conventions.

**Quick reference:**
- `tasks/backlog/` — queued work
- `tasks/active/` — in progress  
- `tasks/done/` — completed
- `tasks/inbox/` — stakeholder requests to triage
- `docs/ROADMAP.md` — milestones and priorities
- `docs/DECISIONS.md` — architecture decision log
- `docs/CHANGELOG.md` — shipped work

**Start every session** by reading `docs/ROADMAP.md` to understand current priorities.

Use `/pm:help` to see all available project management commands.
